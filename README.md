# HireHub

## Smart Campus Placement Management System

### Overview

HireHub is a Java-based placement management system designed to automate campus recruitment activities.

### Technologies

- Java
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Git
- GitHub

### Planned Features

- Student Module
- Company Module
- Placement Officer Module
- Admin Module
- Resume Upload
- Job Application
- Interview Scheduling
- Reports
- Notifications

### Future Enhancements

- Spring Boot
- REST API
- Python AI Integration
- Local SLM
